# Bus Tickets Project
_______

### Instructions to run project:

1. Clone repo.
2. Rename .envs/.postgres.example and edit
3. Rename .envs/.django.example and edit

4. Exec
```python
docker-compose build
docker-compose up -d
```

5. python manage.py migrate authtoken
